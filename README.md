# 基于深度学习的手写数字识别系统 python1025

## 项目概述

本项目是一个基于卷积神经网络（CNN）的手写数字识别系统，采用Python编程语言，并支持PyTorch和TensorFlow两种深度学习框架。系统实现了高精度的识别分类，在MNIST和QMNIST测试集上的准确率可以达到99%。

## 技术栈

- 编程语言：Python
- 环境管理：Anaconda
- 深度学习框架：PyTorch / TensorFlow
- 图像处理库：PIL
- 数据可视化库：matplotlib

## 功能模块

以下是项目的主要功能模块：

1. 数据预处理和增强
2. 数据加载
3. 超参数设置（epochs、batch_size、learning rate）
4. 模型训练
5. 训练记录保存
6. 损失和准确率曲线图绘制
7. 模型评估
8. 混淆矩阵绘制
9. 分类报告表打印
10. 模型应用
11. UI界面实现

## 系统角色

系统主要包括以下角色：

- 数据集：提供用于训练和测试的图片数据。
- 模型：基于CNN架构，用于手写数字的识别。
- 用户界面（UI）：用户可导入图片进行预测，或手绘数字进行识别。

## 运行环境

- 支持的编译器：PyCharm、VSCode、Jupyter Notebook
- Python环境：通过Anaconda管理

## 部署步骤

1. 安装Anaconda及Python环境。
2. 选择PyTorch或TensorFlow作为深度学习框架。
3. 克隆或下载项目代码。
4. 安装所需依赖库。
5. 运行代码进行模型训练和测试。

## 目录结构

```
project/
│
├── dataset/               # 数据集目录
│
├── src/                   # 源代码目录
│   ├── models/            # 模型定义文件
│   ├── training/          # 训练脚本和配置文件
│   ├── utils/             # 辅助工具和函数
│   ├── ui/                # 用户界面代码
│   └── main.py            # 主程序入口
│
├── trained_models/        # 存放训练好的模型
│
└── visualization/         # 可视化图表和混淆矩阵等
```

- 高准确度：在标准手写数字数据集上的准确率高达99%。
- 灵活选择：支持两种主流的深度学习框架。
